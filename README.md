# Smart File Organizer

A simple, locally-hosted web application built with Python and Flask that helps you clean up and organize messy folders on your computer. It scans a target directory and automatically sorts files into categorized folders (Images, Documents, Videos, etc.) based on their file extensions.

## Features

* **User-Friendly Web Interface:** Easy-to-use frontend built with HTML, CSS, and JavaScript.
* **Smart Sorting Algorithm:** Automatically detects file types and moves them into designated category folders.
* **Safe Local Processing:** Runs entirely on your local machine. No files are uploaded to the internet.
* **Error Handling:** Provides real-time feedback if a directory doesn't exist or if an error occurs.

## Technologies Used

* **Backend:** Python, Flask
* **Core Logic:** Python Standard Library (`os`, `shutil`)
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
